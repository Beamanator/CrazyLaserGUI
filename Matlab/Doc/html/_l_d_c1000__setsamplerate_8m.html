function LDC1000_setsamplerate ( in  sport,
in  F 
)

Set the sample rate.

Actual sample rate depends on MSP430 clock and clock div factor.

Example:

* Fser=LDC1000_SetSamplerate(sport,10000);
*

set sample rate to 10000 Hz

Parameters
sportserial port object
FSample rate in Hz
Return values
Actualsample rate in Hz
Remarks
The serial port object is removed from the workspace and deallocated from memory. This function does not handle errors.
