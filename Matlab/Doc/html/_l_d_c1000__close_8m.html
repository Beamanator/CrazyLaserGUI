function LDC1000_close ( in  sport)

Close serial port.

Close and free the serial port for access by other programs.

Example:

* ret=LDC1000_close(sport);
*
Parameters
sportthe serial port object to close
Return values
1if success
Remarks
The serial port object is removed from the workspace and deallocated from memory. This function does not handle errors.
