Read Streamed Data.

Sends command to stop streaming, then flushes the serial buffer by consuming data.

Example:

* [Rp, T]=LDC1000_streamdata(sport, 0, 2^20, 2^13);
*

Acquire 2^20 samples of Rp and L. Display 2^13 samples at a time.

Parameters
sportserial port object
mode1->continous 0-> single shot. If continous mode is selected, press any key when the matlab figure (plot) window is selected. This will halt data aquisition
Nnumber of points to acquire (decimal)
Mnumber of points to display (decimal)
Return values
proximitydata, frequency counter data
Remarks
The serial port object is removed from the workspace and deallocated from memory. This function does not handle errors.

Scan for keyboard activity without interrupting event stream.

MATLAB does not have a function that scans for keyboard activity without interrupting the event stream.

However, if your program involves a figure window, you can utilize the KeyPressFcn property. The 'KeyPressFcn' is called when a key is pressed with an active figure window. You can set this function to change the state of a flag that ends a loop. Copy the following functions to a MATLAB file, and execute the MATLAB file.

The code will display "looping" in the MATLAB Command Window until a key is pressed when the figure window is active
