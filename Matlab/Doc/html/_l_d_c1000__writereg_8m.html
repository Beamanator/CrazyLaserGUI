Write register.

Write the contents of a register

Example:

* data=LDC1000_writereg(sport,11,0);
*

write register 11 with 0, data is data written (expect 0)

Parameters
sportserial port object
regregister address in decimal
wdatadata to write in decimal
Return values
datawritten in decimal
Remarks
The serial port object is removed from the workspace and deallocated from memory. This function does not handle errors.
